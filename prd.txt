Project Name: MindVault Telegram Bot
1. Overview

MindVault is a Telegram-based personal knowledge vault bot that allows users to save links, articles, videos, and text directly from Telegram chats. The bot automatically extracts metadata, generates filenames, assigns tags using rule-based logic (no AI, no external APIs), and stores the content in a structured format.

The bot is designed to run fully serverless using Telegram webhooks on Vercel or Netlify, with zero mandatory cost. All data remains user-owned and exportable as Markdown.

2. Core Features
2.1 Content Saving

The bot accepts:

URLs (articles, blogs, YouTube links)

Plain text notes

Forwarded Telegram messages

For each input, the bot:

Extracts title and description

Detects content type (article, video, note)

Generates a filename

Applies automatic tags

Stores a Telegram message deep link for reference

2.2 Auto Filename & Tagging

Filename format:

<primary-keyword>-<secondary-keyword>-YYYY-MM-DD.md


Tagging logic (rule-based):

Domain-based (youtube â†’ #video, medium â†’ #article)

Each saved item includes:

Storage is serverless-compatible (JSON / KV / optional database).

5. Export & Backup System
5.1 Markdown Export

Users can export all saved data as clean Markdown files, organized tag-wise into folders, with an auto-generated INDEX.md.

5.2 Daily Automatic Backup

Runs once every 24 hours

Triggered via Vercel Cron

Generates a ZIP backup

Filename includes date and time

Example:

mindvault-backup-2026-02-06_23-59-IST.zip

6. Backup Delivery Options (User Choice)
Option A: Telegram (Default)

Backup ZIP is sent directly to the user as a Telegram document

If size exceeds Telegram limits, a message is sent instead

Option B: Google Drive (Optional)

User can optionally log in via Google OAuth

Backups uploaded to:

/MindVault/daily-backups/


Drive login is not mandatory

Bot works fully without Google Drive

7. Commands
